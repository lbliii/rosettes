v0.3.0 Released: January 2, 2025 Initial public release of Rosettes, extracted from the Bengal static site generator. Highlights 55 language lexers — Hand-written state machines for Python, JavaScript, Rust, Go, and 51 more O(n) guaranteed — Linear time complexity, no ReDoS vulnerabilities Thread-safe — Designed for Python 3.14t free-threading Pygments compatible — Drop-in CSS class compatibility Parallel API — highlight_many() for concurrent processing Features Core API highlight() — Generate HTML with syntax highlighting tokenize() — Get raw tokens for custom processing highlight_many() — Parallel highlighting for multiple blocks tokenize_many() — Parallel tokenization Registry get_lexer() — Get lexer by name or alias list_languages() — List all supported languages supports_language() — Check language support Styling Semantic CSS classes (.syntax-keyword, .syntax-function, etc.) Pygments-compatible classes (.k, .nf, etc.) Line highlighting with hl_lines Line numbers with show_linenos Languages 55 languages supported: Category Languages Core Python, JavaScript, TypeScript, JSON, YAML, TOML, Bash, HTML, CSS, Diff Systems C, C++, Rust, Go, Zig JVM Java, Kotlin, Scala, Groovy, Clojure Scripting Ruby, Perl, PHP, Lua, R, PowerShell Functional Haskell, Elixir Modern Dart, Julia, Nim, Gleam, V, Mojo Config INI, Nginx, Dockerfile, Makefile, HCL And more... Swift, SQL, GraphQL, Markdown, XML, Protobuf, etc. Installation pip install rosettes Requires Python 3.14+. Quick Start from rosettes import highlight html = highlight(&quot;def hello(): print(&#x27;world&#x27;)&quot;, &quot;python&quot;) Known Limitations HTML output only (no LaTeX, RTF, terminal formatters) 55 languages (vs Pygments' 500+) Python 3.14+ required Links Documentation GitHub Repository PyPI Package
