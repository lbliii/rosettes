Pygments Themes Rosettes supports Pygments-compatible CSS classes, allowing you to use any existing Pygments theme. Enable Pygments Classes from rosettes import highlight html = highlight(code, &quot;python&quot;, css_class_style=&quot;pygments&quot;) This generates HTML with Pygments class names: &lt;div class=&quot;highlight&quot; data-language=&quot;python&quot;&gt; &lt;pre&gt;&lt;code&gt;&lt;span class=&quot;k&quot;&gt;def&lt;/span&gt; &lt;span class=&quot;nf&quot;&gt;hello&lt;/span&gt;... Using a Pygments Theme Option 1: CDN Popular themes are available via CDN: &lt;!-- Monokai --&gt; &lt;link rel=&quot;stylesheet&quot; href=&quot;https://cdnjs.cloudflare.com/ajax/libs/highlight.js/11.9.0/styles/monokai.min.css&quot;&gt; &lt;!-- Dracula --&gt; &lt;link rel=&quot;stylesheet&quot; href=&quot;https://cdnjs.cloudflare.com/ajax/libs/highlight.js/11.9.0/styles/dracula.min.css&quot;&gt; Note Note highlight.js themes work with Pygments classes since they use the same class names. Option 2: Generate from Pygments If you have Pygments installed: pygmentize -S monokai -f html &gt; monokai.css Option 3: Copy a Theme Here's Monokai as an example: .highlight { background: #272822; } .highlight .c { color: #75715e } /* Comment */ .highlight .k { color: #66d9ef } /* Keyword */ .highlight .n { color: #f8f8f2 } /* Name */ .highlight .o { color: #f92672 } /* Operator */ .highlight .p { color: #f8f8f2 } /* Punctuation */ .highlight .s { color: #e6db74 } /* String */ .highlight .m { color: #ae81ff } /* Number */ .highlight .nf { color: #a6e22e } /* Name.Function */ .highlight .nc { color: #a6e22e } /* Name.Class */ .highlight .nd { color: #a6e22e } /* Name.Decorator */ .highlight .nb { color: #f8f8f2 } /* Name.Builtin */ Popular Themes Theme Style Background Monokai Dark #272822 Dracula Dark #282a36 One Dark Dark #282c34 Solarized Dark Dark #002b36 Solarized Light Light #fdf6e3 GitHub Light #ffffff GitHub Dark Dark #0d1117 Container Class When using Pygments style, the container class changes: # Semantic style (default) html = highlight(code, &quot;python&quot;) # &lt;div class=&quot;rosettes&quot;&gt; # Pygments style html = highlight(code, &quot;python&quot;, css_class_style=&quot;pygments&quot;) # &lt;div class=&quot;highlight&quot;&gt; # Custom container html = highlight(code, &quot;python&quot;, css_class_style=&quot;pygments&quot;, css_class=&quot;codehilite&quot;) # &lt;div class=&quot;codehilite&quot;&gt; Migration from Pygments If you're switching from Pygments to Rosettes: Keep your existing CSS theme Use css_class_style=&quot;pygments&quot; No CSS changes needed # Before (Pygments) from pygments import highlight as pyg_highlight from pygments.lexers import PythonLexer from pygments.formatters import HtmlFormatter html = pyg_highlight(code, PythonLexer(), HtmlFormatter()) # After (Rosettes) from rosettes import highlight html = highlight(code, &quot;python&quot;, css_class_style=&quot;pygments&quot;) Next Steps Custom Themes — Build your own theme Migration Guide — Full migration walkthrough
